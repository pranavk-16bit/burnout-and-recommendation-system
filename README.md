# burnout-and-recommendation-system
An ML-based system to predict student burnout levels using behavioral, academic, and lifestyle data. This project aims to provide insights and future recommendations for improving student mental health.

DAY 1:
Loaded dataset using Pandas
Cleaned column names
Converted target variable (burnout_level) into numeric form
Applied One-Hot Encoding for categorical features
Split dataset into training and testing sets
Built a baseline Logistic Regression model
Evaluated performance using classification metrics
